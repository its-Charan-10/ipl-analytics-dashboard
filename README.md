# IPL Analytics Dashboard

An interactive data analytics dashboard built using **Python, Pandas, and Streamlit** to analyze IPL player and team performance.

---

## Features

-  Player Performance Analysis  
-  Team-wise Run & Strike Rate Analysis  
-  Custom **Impact Score** Metric  
-  Player Comparison Tool  
-  Player Search Functionality  
-  Clean & Interactive Dashboard UI  

---

## Impact Score Formula

Impact Score is a custom metric used to evaluate player performance:

Impact Score = (Runs × Strike Rate) / 100

 Combines:
- Runs (consistency)
- Strike Rate (aggression)
